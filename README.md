# Bonsai Pricing Page – Single Page Clone

A fully responsive and dynamic single-page clone of the Bonsai Pricing page.  
This project was created to practice real-world front-end implementation, UI/UX structure, Tailwind styling, and interactive UI components.

Original page (archived version):  
https://web.archive.org/web/20221117021714/https://www.hellobonsai.com/pricing

## Overview
This project recreates the layout and interactions of the Bonsai pricing page using **HTML**, **Tailwind CSS**, and **JavaScript**.  
The goal was to analyze an existing UI and reproduce its structure, styling approach, and dynamic behavior.


##  Features
-  **Fixed Header** with responsive layout  
-  **Dropdown Menu**  
-  **Accordion Sections** (expand/collapse on click)  
-  **Fully Responsive Design** (mobile → desktop)  
-  **Masonry-Style Footer Layout**  
-  **Dynamic Interactions** using JavaScript  
-  **Tailwind CSS (via npm)** for styling  
-  **Component-based utility classes** for cleaner structure  

##  Technologies Used
- **HTML**
- **Tailwind CSS**
- **JavaScript**
- **Git / GitHub** (version control)


##  How to Run the Project
1. Clone the repository  
2. Run `npm install` to install Tailwind  
3. Use `npm run dev` or your Tailwind build script
